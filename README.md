# Eenvoudige Console Applicatie voor Naambeheer

Ontwikkel een eenvoudige C# console applicatie die gebruikers in staat stelt om een lijst met namen in te voeren en deze vervolgens weer te geven. De applicatie moet de gebruiker vragen hoeveel namen er ingevoerd moeten worden, vervolgens de namen één voor één inlezen en opslaan in een array. Tot slot moet de applicatie alle ingevoerde namen op de console weergeven. Dit project richt zich op basis C# syntax, array-manipulatie en console input/output. Er is geen foutafhandeling vereist voor deze oefening.

## Opdracht

Implementeer de gevraagde functionaliteit.
